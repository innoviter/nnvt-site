# nnvt-site

## Tech Stack

- Astro
- Tailwind CSS
- Docker Compose (for local development AND production)

## Key Features

- Component Library
- Mobile First (with mobile, tablet and desktop viewports)
- Static Site Generation (SSG)
- Global Styles
- Dark and Light Mode
- Zero JavaScript
- Internationalization (route-based i18n)
  - example.com/de/
  - example.com/en/
- Integration websitecarbon.com
- Integration greenweb.org Badge (Green Web Foundation)
- File-Based Routing
- Later: Matomo integration
- Later: Content Collections
- Later: JavaScript for Cookie Consent Dialog
- Later: Vue.js
- Later: SEO
- Later: Image Optimization
- Later: Directus Integration
